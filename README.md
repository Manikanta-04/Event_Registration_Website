# ⚡ Ben 10 Universe Summit — Event Registration Website

> *It's Hero Time!* A visually immersive, Ben 10–themed event registration platform featuring animated 3D Omnitrix, alien selector, mission schedule, and a fully functional registration form — built with pure HTML, CSS & JavaScript.

---

## 🟢 Project Description

<!-- 200 characters -->
Ben 10–themed event registration site with a 3D animated Omnitrix, alien identity selector, event schedule timeline, and styled registration form. Built using HTML, CSS & JavaScript.

---

## 🚀 Live Demo

> Open `index.html` in any modern browser — no server required.

---

## 🛸 Features

- **3D Animated Omnitrix** — Floating, rotating hero badge with pulsing neon-green glow, spinning outer ring, and CSS 3D transforms
- **Alien Identity Selector** — Choose from 6 iconic aliens (Heatblast, XLR8, Diamondhead, Grey Matter, Four Arms, Wildvine)
- **Event Stats Counter** — Animated counters for Heroes, Alien Forms, Hours & Days
- **Mission Schedule Timeline** — Scrollable event program with hover interactions
- **Registration Form** — Full form with pass tier selection, alien pick, and success overlay
- **Particle Effects** — Floating green particles, twinkling stars, and grid-scroll background
- **Mouse-Reactive Tilt** — 3D CSS perspective tilt on alien cards
- **Fully Responsive** — Works on mobile, tablet, and desktop

## 🧬 Tech Stack

| Technology | Usage |
|------------|-------|
| HTML5 | Structure & semantics |
| CSS3 | 3D transforms, keyframe animations, clip-path, glassmorphism |
| Vanilla JavaScript | Counter animations, form handling, particle generation |
| Google Fonts | Orbitron (sci-fi display) + Rajdhani (body) |
